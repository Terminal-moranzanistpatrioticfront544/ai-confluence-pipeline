## 📌 What this app does

ai-confluence-pipeline helps you turn technical notes into useful team work.

It can:
- analyze technical content with AI
- send results to Confluence
- create Jira tasks from the same work
- keep your process in one place
- save time for tech leads and project owners

This app fits teams that want a simple way to move from review work to shared documentation and task tracking.

## 🖥️ What you need

Before you start, make sure you have:
- a Windows PC
- internet access
- a web browser
- access to Confluence
- access to Jira

For best results, use:
- Windows 10 or Windows 11
- at least 8 GB of RAM
- enough free disk space for the app and its files
- a stable network connection

## 🚀 Install on Windows

After you download the file:

1. Open the Downloads folder
2. Find the file you just downloaded
3. If it is a `.zip` file, right-click it and choose Extract All
4. Open the extracted folder
5. If it is an `.exe` or `.msi` file, double-click it
6. Follow the on-screen steps
7. If Windows asks for permission, choose Yes

If the app comes in a folder after extraction, keep all files together. Do not move only one file out of the folder.

## ▶️ Run the app

After install, open the app from:
- the Start menu
- your desktop shortcut
- the folder where you extracted it

If the app opens in a browser, leave that window open while you work.

If Windows SmartScreen appears:
1. Select More info
2. Select Run anyway

This can happen with newly released apps that are not yet common on Windows systems.
